Login Widget for BracketPress

== Description ==

Login & Registration Widget for BracketPress

Based on "Tabbed Login Widget" by Vivek Marakana

== Installation ==

= First time installation instructions =

1. Install the Plugin
1. Activate it
1. Drag the widget to your side bar
1. Configure options

== Frequently Asked Questions ==

= Is BracketPress Required to use this Widget?? =

Yes. This Plugin requires Core BracketPress functionality to work properly.

= How do I enable the Registration functionality? =

Settings > General > tick "Anyone Can Register"
Then set "Registration Tab" to "Tab Enabled"
